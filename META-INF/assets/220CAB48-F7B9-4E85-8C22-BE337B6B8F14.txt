AndroRat.
Шпион для Android - AndroRat + AndroRat Binder
Программа AndroRat, это RAT для смартфонов базирующихся на платформе Android. Прога написана на Java и предоставляет следующие возможности для хакера:

-Получить с устройства жертвы список контактов и всю информацию о них;
-Получить списки звонков из журналов истории;
-Получить все сообщения сохраненные на телефоне;
-Получить текущее местоположение по GPS;
-Мониторинг входящих сообщений в реальном времени;
-Мониторинг состояния телефона в реальном времени (входящие звонки, пропущенные звонки и т.д.);
-Захват изображения с камеры;
-Стриминг звука с микрофона (или с другого источника);
-Стриминг видео с камеры;
-Отправить SMS-сообщение;
-Осуществить звонок;
-Открыть URL в стандартном браузере;
-Следить за местонахождением телефона по GPS данным;
-Заставить телефон завибрировать и др.
Но стоит отметить тот факт что данная программа многофункциональная ее можно настроить для работы через интернет но также и в локальной домашней сети WI-FI.
Сейчас я постараюсь подробно описать и привести примеры использования этой программы.
Так как информации очень мало в интернете. Да и я сам не программист, пусть простят меня хакеры, все что я изложу в этом эпитете, я на собственном примере нарывал юзая данную программу.
Для начала скачиваем пакет программы, распаковываем ее в папку созданную на рабочем столе.
В пакет входит.

AndroRat Binder.- Программа формирования и упаковки билда RAT. Она как упаковывает отдельным apk приложением в двух вариантах. 1 вариант простое приложение, 2 вариант скрытое приложение, его не видно в панели управления телефоном. И самый интересный 3 вариант, этот билд RAT присоединяется к любому apk приложению.
В папке AndroRAT непосредственно сама программа для мониторинга билда RAT который устанавливается в телефон.
В папке framework непосредственно сам билд RAT который с помощью программы AndroRat Binder упаковывается в apk файл для установки на телефон.

Итак скачали, распаковали. С чего начать? 
А начнем с приготовления и упаковки в apk билда RAT.
Для начала мы сформируем самый простой вариант билда для работы в локальной домашней сети WI-FI. Если ваш wi-fi роутер или адаптер раздающий сеть интернета имеет постоянный IP адрес проблем не возникнет ни каких. Так как билд установленный в телефон будет связываться непосредственно по локальному IP адресу и вести полный контроль над телефоном. 
Запускаем AndroRat Binder.
<img>2762408F-1952-4D16-B008-700E14428029.png</img>
Для начала нам необходимо его настроить так как нам нужен apk билд для работы в wi-fi сети в 1 окне красным выделен Build+Bind- в этом режиме производится упаковка билда и присоединении его к любому apk приложению.
Но в данный момент нам необходимо в этом окне только ввести IP адрес и номер PORT. В моем случае я вводил 192.168.0.1 это ip моего wi-fi адаптера и порт 80.
При переходе в окно No-IP вводить ничего не нужно, так как данная опция используется при формировании билда для работы через интернет, об этом я изложу позже.
 Переходим сразу в окно Build.
<img>6E978108-E35F-436E-B494-E214DA93E4D6.png</img>
В нем мы сразу увидим введенные в первом окне IP и PORT.
В моем случае я вводил 192.168.0.1 это ip моего wi-fi адаптера и порт 80.
В окошке APK TITLE изначально введено имя папки с билдом RAT
Framework вы можете ввести любое желаемое имя.
Я вводил имя system чтобы приложение не выдавало себя в диспетчере приложений за шпиона. Если вы включите опцию Hidden то ваше приложение будет скрытым и его не будет видно в панели приложений телефона.
И после всех не долгих манипуляций кликаем на GO и получаем в папке с программой нужный нам  system.apk файл.Который нам нужно будет потом установить на телефон жертвы.
Также можно просто присоединить RAT билд к любому apk приложению. Я использовал простую игру DalmaxCheckers.apk.
Что нам нужно сделать. Кидаем приложение DalmaxCheckers.apk в папку с программой. 
Запускаем AndroRat Binder.
Также в главном окне вводим IP и PORT. В моем случае я вводил 192.168.0.1 это ip моего wi-fi адаптера и порт 80.
Кликаем Browse и выбираем нужный нам файл DalmaxCheckers.apk. После кликаем GO.
По завершению процесса компиляции в папке с программой мы получаем  FINISHED.apk. 
Непосредственно уже готовый RAT для установки на телефон.
После установки приложения на телефон, подключаем его к свой сети WI-FI.
Открываем папку AndroRAT.
<img>14B31803-72B5-48ED-AE8E-845E1F78F8FA.png</img>
И запускаем AndroRat Java. Если билд RAT сделан правильно и телефон соединен с вашим роутером.
В окне программы AndroRat Java вы получите соединение с телефоном для управления.
<img>699FB2C7-15C6-4D21-9081-7304823DE5F8.png</img>
Также хотелось бы отметить маленькую деталь для более конкретной привязки RAT билда к вашей WI-FI сети.
В папке с программой AndroRat Binder находится папка Framework в этом папке, папка res в этой папке папка layout в ней лежит файл main. Открываем его в Notepad++ там по умолчанию вбит IP адрес 192.168.0.12 и порт 9999. Заменяем эти данные на свои. В моем случае я вводил 192.168.0.1 это ip моего wi-fi адаптера и порт 80.
<img>DE63D5A3-0F26-44D6-9AD4-D1640C32043A.png</img>
Итак я изложил простейший вариант использования этой программы.
Теперь рассмотрим более сложный вариант ее работы, а именно вариант работы через интернет. Почему он сложный, суть в том что нам нужен постоянный IP адрес.
Для этого нам нужно зарегистрироваться на сайте https://www.noip.com/
Чтобы получить бесплатный Host name.
Host name это ваш логин, по сути логин который не изменяясь привязывает IP адреса.
Также скачать с сайта программу No-IP DUC где и будет использоваться ваш Host name.
Итак начнем.
Мы зарегистрировались на сайте https://www.noip.com/ получили Host name, скачали и запустили No-IP DUC
Далее переходим к созданию нужного нам RAT билда.
Запускаем AndroRat Binder.
<img>563CB2F3-CD36-439E-A649-07D0A82C5DE5.png</img>
В окне IP вводим имя своего Host name.
В окне PORT номер порта которым пользуется ваш 
Host name. Узнать порт вы можете через программу 
No-IP DUC.
Далее переходим на вкладку No-IP.
В поле Username вводим свой адрес электронной почты который был зарегистрирован на сайте https://www.noip.com/
В поле Password вводим пароль, тот же который был применен на сайте noip.com 
В поле Hostname вводим имя своего хоста.
И кликаем Update для проверки легитивности введенных данных. Если вы ввели правильные данные, внизу Status
Зеленым свето появится надпись Update Successful.
Это означает что теперь вы можете перейти далее для создания apk билда RAT. Переходим во вкладку Build и создаем нужный нам файл.
Или переходим во вкладку Build+Bind и создаем присоединенный к apk приложению RAT билд.
Далее все как говориться ПО СТАРЫМ чертежам.
<img>426F4148-DE96-489B-BEED-32601981CBC3.png</img>
Устанавливаем билд на телефон.
Запускаем на компьютере No-IP DUC
И запускаем AndroRat Java. Если билд RAT сделан правильно и телефон в сети интернета.
В окне программы AndroRat Java вы получите соединение с телефоном для управления.
